## Description
#### This is my capstone project for [neuefische](https://www.neuefische.de/en) web development bootcamp.

I've created Kanzepts to help you plan your meals ahead of time. In addition to a nice overview of your planned recipes, Kanzepts provides you with inspiration for meals to cook, allows you to mark your favorite recipes and keeps a history of past meals. This way, you will never lose track of your favorite dish and can make sure to schedule it at least once per week. ðŸ˜‰
